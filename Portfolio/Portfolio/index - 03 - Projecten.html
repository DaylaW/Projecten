Projecten

Stage project #01

Aan het einde van elk blok (3 maanden) krijgen we een project die we binnen 2 weken moeten afronden.
In het eerste project kregen we de opdracht om met 10 bedrijven af te spreken om een paar vragen te stellen over hoe zij werken met stagiaires.
Dit was om ons voor te bereiden op een stage zoeken in het derde blok.

Stage project #02

In dit project kregen we verschillende opdrachten.

Voor de eerste opdracht moesten we een website maken om Haarlem te promoten voor toeristen. De eisen van de website waren dat er meerdere pagina's zijn, in het Engels en Nederlands te bekijken is en 4 bezienswaardigheden bevatten waar een stukje over verteld wordt.

In de tweede opdracht maakten we in excel een bestelformulier voor het "bedrijf" Impressive Red Apple. Hierin moest de klant kunnen kiezen uit verschillende hardware onderdelen en daarvoor de totaal prijs kunnen bekijken.

Ziggo website

In dit project heb ik de startpagina van de website van Ziggo nagemaakt. We kregen een gedetaileerde uitleg via een presentatie, daarin stonden alle stappen die we moesten volgen. Bij het begin van de website stond er stap voor stap wat we moesten opschrijven in de code, maar na mate de website completer werd moesten we steeds meer van de codering zelf invullen.

Online Portfolio

Dit portfolio is een project

De opdracht is om een persoonlijk online portfolio te maken in HTML en CSS waarin je jezelf en je gemaakte producten online presenteert. De site moet ook vanaf het internet bereikbaar zijn.

Dit portfolio wordt gebruikt door de gehele opleiding. Gaan de weg zal ik de inhoud van het portfolio aanpassen en uitbreiden met nieuwe ervaringen en producten. Wanneer je de opleiding afrond presenteer je ook de laatste versie van het online portfolio
